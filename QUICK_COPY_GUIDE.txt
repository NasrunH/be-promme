═════════════════════════════════════════════════════════════════════════════

src/utils/pagination.js
├─ parsePagination() - Extract page/limit from query
├─ parseFilters() - Extract allowed filters from query
├─ parseSearch() - Extract and validate search term
├─ parseSort() - Extract and validate sort field
├─ parseRangeFilter() - Extract min/max numeric filters
├─ parseDateRangeFilter() - Extract date range filters
├─ formatPaginationResponse() - Format response with metadata
├─ buildAggregationPipeline() - For complex queries
└─ createMultiFieldSearchFilter() - For multi-field search

STANDARD QUERY PARAMETERS (All endpoints)
═════════════════════════════════════════════════════════════════════════════

?page=1                      # Page number (default 1)
&limit=10                    # Items per page (default 10, max 100)
&search=keyword              # Search term (varies by endpoint)
&filter_field=value          # Filter by specific field
&field_min=100               # Range filter minimum
&field_max=1000              # Range filter maximum
&sort=field                  # Sort ascending
&sort=-field                 # Sort descending
